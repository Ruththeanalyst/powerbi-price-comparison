# powerbi-price-comparison
Power BI dashboard comparing product prices across stores to identify cheaper options

## Project Overview
- Tracks the **total price of products** in different stores (e.g., Walmart, Target, Costco).  
- Shows the **best store for each product**.  
- Visualizes:  
  - **Bar charts** for total price by store and by product  
  - **Comparison of prices** for specific products across stores  

## Skills Demonstrated
- Power BI Desktop  
- Data visualization with bar charts  
- Conditional formatting and chart customization  
- Dashboard layout and design
